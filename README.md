# 🎮 Tic Tac Toe Android App

A simple and interactive **Tic Tac Toe game** built as an Android application using **Android Studio**.
This project demonstrates game logic, UI design, and event handling in mobile app development.

---

## 📱 Features

* 🎯 Two-player mode (Player X vs Player O)
* 🧠 Win and draw detection logic
* 📲 User-friendly Android UI
* ⚡ Lightweight and fast performance

---

## 🛠️ Tech Stack

* Language: Java 
* IDE: Android Studio
* UI: XML Layouts/Adobe
* Platform: Android

---



## 📥 How to Download the Project

### 🔹 Method 1: Download ZIP (Easy)

1. Go to your GitHub repository
2. Click **Code**
3. Click **Download ZIP**
4. Extract the ZIP file

---

### 🔹 Method 2: Clone using Git (Recommended)

```
git clone https://github.com/your-username/your-repo-name.git
```

---

## ▶️ How to Run the App

1. Open **Android Studio**
2. Click on **Open**
3. Select the project folder
4. Let Gradle build the project
5. Connect an Android device or start an emulator
6. Click ▶️ **Run**

---

## 📦 APK Installation (Optional)

If APK is provided:

1. Transfer APK to your Android device
2. Enable **Install from Unknown Sources**
3. Tap the APK file and install
